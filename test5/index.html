Информация

Перед вами на странице 3 поля

- 1 поле: Для ввода длинны массива, должна быть больше 0.

- 2 поле: Для ввода минимума массива, любое целое число.

- 3 поле: Для ввода максимума массива, любое целое число.

После ввода всех данных нужно нажать на кнопку "Создать массив".

После нажатия кнопки на экране должен появиться массив, который не будет изменяться и который будет изменяться.

После того, как массивы появились на экране, нужно нажать на кнопку "Сортировать".

После того, как массив отсортировался, появиться всплывающее окно.

Так же на странице есть кнопка "Начать заного", которая сбросит массив и удалит блоки с массивами с экрана.
